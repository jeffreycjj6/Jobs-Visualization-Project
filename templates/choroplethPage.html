Color Shading Scale Explanation:

The color shading scale indicates a divisor for the max number of job entries for that type. It is used to allow more states to show color.

Higher numbers mean lower overall maximums, causing colors to be scaled down and more similar between states.

This is done to prevent states such as California from consistently raising color thresholds and causing less dominate states from having any color.

Typically, lower values should be used for less popular jobs and higher values for more common jobs to get better visual map information at the cost of the true listings count.

A value of 1 is the original unaltered scale, where each state is colored exactly as it should be based on how many jobs of that type it truly has.
